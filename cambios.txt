================================================================================
                        COMANDO PARA VERIFICAR MD5
================================================================================

Para verificar que un archivo no ha sido modificado, calcular su MD5:

Windows (PowerShell):
Get-FileHash -Algorithm MD5 archivo.py

Linux/Mac:
md5sum archivo.py

Python:
python -c "import hashlib; print(hashlib.md5(open('archivo.py','rb').read()).hexdigest())"

Comparar el resultado con el MD5 registrado en este documento.
Si coinciden: ✅ Archivo íntegro
Si NO coinciden: ⚠️ Archivo modificado
