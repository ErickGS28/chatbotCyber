  VERSIÓN 3.0 - CHATBOT CON CIFRADO SIMÉTRICO + SHA256 + MD5

Esta es la versión actual con todas las características de seguridad:
- Cifrado simétrico AES (Fernet)
- Hash SHA256 para verificación de integridad de mensajes
- Cálculo de MD5 para verificación de integridad de archivos

════════════════════════════════════════════════════════════════
📋 INSTRUCCIONES DE EJECUCIÓN:
════════════════════════════════════════════════════════════════

1️⃣  Abrir terminal y navegar a esta carpeta:
    cd versiones/v3.0_con-sha256-md5/chatbot-simetrico

2️⃣  Instalar dependencias:
    pip install -r requirements.txt

3️⃣  Ejecutar el servidor:
    python servidor.py

4️⃣  Abrir navegador en:
    http://localhost:5001

5️⃣  Ingresar tu nombre y comenzar a chatear
