════════════════════════════════════════════════════════════════
       VERSIÓN 2.0 - CHATBOT CON CIFRADO ASIMÉTRICO (RSA)
════════════════════════════════════════════════════════════════

Esta versión implementa cifrado asimétrico usando RSA.
Los mensajes se cifran con llave pública y descifran con llave
privada, pero aún NO tiene verificación de integridad con SHA256.

════════════════════════════════════════════════════════════════
📋 INSTRUCCIONES DE EJECUCIÓN:
════════════════════════════════════════════════════════════════

1️⃣  Abrir terminal y navegar a esta carpeta:
    cd versiones/v2.0_cifrado-separado/chatbot-asimetrico

2️⃣  Instalar dependencias:
    pip install -r requirements.txt

3️⃣  Ejecutar el servidor:
    python servidor.py

4️⃣  Abrir navegador en:
    http://localhost:5002

5️⃣  Ingresar tu nombre y comenzar a chatear

════════════════════════════════════════════════════════════════
✨ CARACTERÍSTICAS DE ESTA VERSIÓN:
════════════════════════════════════════════════════════════════

✅ Chat multiusuario en tiempo real
✅ WebSockets para comunicación instantánea
✅ Cifrado asimétrico con RSA-2048
✅ Los mensajes se cifran antes de logging
✅ Usa llaves pública/privada

❌ NO tiene hash SHA256
❌ NO tiene verificación de integridad
❌ NO tiene cálculo de MD5 de archivos
