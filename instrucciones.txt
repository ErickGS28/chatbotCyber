INSTRUCCIONES PARA EJECUTAR LEXBOT
===================================

⚠️  CONFIGURAR TU API KEY DE GEMINI:
1. Ve a: https://aistudio.google.com/app/apikey
2. Crea una nueva API Key
3. Abre config.py y reemplaza la API key existente con la tuya
4. Guarda el archivo

🚀 EJECUTAR EL CHATBOT:
1. Instalar dependencias:
   pip install -r requirements.txt

2. Probar conexión:
   python test_gemini.py

3. Ejecutar el chatbot:
   python servidor_gemini.py

4. Abrir navegador:
   http://localhost:5001


⚠️  SEGURIDAD DE API KEYS:
- NUNCA compartas tu API key con otros
- Cada desarrollador debe crear su propia API key
- Las API keys tienen límites de uso y costos


═══════════════════════════════════════════════════════════════
VERSIÓN ANTIGUA - CONSOLA (Solo si necesitas usar la terminal)
═══════════════════════════════════════════════════════════════

1. Ejecutar servidor:
   python servidor.py

2. En otra terminal, ejecutar cliente:
   python cliente.py

3. Escribir "salir" para desconectar

SOLUCIÓN DE PROBLEMAS:
- Si puerto ocupado: cambiar números de puerto en los archivos
- Para detener: Ctrl+C en la terminal
